
Porfololio side 1:

Denne portfolio siden (MyPortfolio) lages automatisk. Her kan aksjer legges in, oppdateres eller slettes. Siden viser på toppen oppsumering av denne porteføljen. Under ser informasjon om de spesifike aksjene som hvor mange ganger de betaler en dividend, aksjer som kan kjøpes med reinvistering (DRIP). Så på slutten oppsumering av det totale og de siste 24 timene.
